# Устанавливаем Python

1. Загружаем установщик Python:
    
    https://www.python.org/downloads/windows/

2. Выполняем установку. ВАЖНО! Включить настройку "Add python.exe to PATH" в диалоговом окне установки (по умолчанию выключена). Также рекомендуется записать путь к директории, в которую устанавливается Python.

    ![Диалоговое окно установки](https://docs.python.org/3/_images/win_installer.png)

    https://docs.python.org/3/using/windows.html#using-on-windows

3. Если Python уже установлен, но Вам необходимо добавить python.exe в переменную PATH, можно сделать следующее:

     - Выполнить переустановку Python, включив настройку "Add python.exe to PATH" в диалоговом окне установки.

     - Добавить путь к python.exe в переменную PATH вручную, для этого:

        - Откройте "Панель управления", в строке поиска введите: "Изменение системных переменных среды"
        - Откройте диалоговое окно "Свойства системы" нажатием на ссылку "Изменение системных переменных среды"
        - В открывшемся окне нажмите на кнопку "Переменные среды".
        - Найдите переменную "Path" в списке системных переменных и нажмите "Изменить".
        - Нажмите на "Добавить" и введите путь к папке с исполняемым файлом Python (например, "C:\Python38").
        - Нажмите на "ОК" во всех открытых окнах.
